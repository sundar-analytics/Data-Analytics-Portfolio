# Global Data Job Market Analysis Dashboard

## Project Overview
Analyzed 479,000+ global data job listings to identify hiring trends, salary benchmarks, job demand, remote work opportunities, and industry insights using Power BI and Excel.

## Tools & Technologies
- Power BI
- Excel

## Key Features
- Processed and transformed 479K job records
- Built interactive dashboards with dynamic filters
- Created salary trend and demand analysis visualizations
- Implemented drill-through pages for role-level insights

## Key Insights
- Data Engineer was the most in-demand role
- Senior roles offered significantly higher compensation
- Most data jobs were on-site
- LinkedIn was the leading hiring platform
