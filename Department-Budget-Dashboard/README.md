# Department Budget & Financial Analysis Dashboard

## Project Overview
Developed an end-to-end Business Intelligence solution to analyze departmental budgets, expenses, project costs, and financial performance across multiple departments.

## Tools & Technologies
- SQL
- Power BI
- DAX
- Microsoft Excel

## Business Problem
Departments were managing budgets across multiple datasets with limited visibility into expenses, project costs, and budget utilization.

## Key Features
- Consolidated data using SQL joins and CTEs
- Designed a Star Schema data model in Power BI
- Developed custom DAX measures
- Created interactive dashboards
- Built automated risk indicators

## Skills Demonstrated
- SQL
- Power BI
- DAX
- Data Modeling
- Business Intelligence
