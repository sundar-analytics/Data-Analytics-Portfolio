# Regional Sales Performance Dashboard

## Project Overview
This Power BI dashboard was developed to help Sales Managers monitor regional performance, track category-wise target achievement, analyze returns, identify delivery delays, and measure year-over-year business growth.

## Business Problem
The organization operates across multiple regions and product categories. Management needed a centralized dashboard to understand sales performance, target achievement, return trends, and delivery efficiency to support better business decisions.

## Tools Used
- Power BI
- DAX
- Power Query
- Excel
- Data Modeling

## Key KPIs
- Total Sales
- Sales YoY %
- Target Achievement %
- Return Rate %
- Delay %

## Dashboard Features
- Regional Sales Analysis
- Manager-wise Performance Tracking
- Category-wise Target Achievement
- Returns Analysis
- Delivery Delay Monitoring
- Interactive Filters & Slicers

## Business Insights
- Identified top-performing regions
- Tracked category-wise target achievement
- Analyzed return patterns
- Monitored delivery delays
- Supported data-driven decision-making
